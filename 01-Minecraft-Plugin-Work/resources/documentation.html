Scrap Your Gear

Scrap Your Gear is a Minecraft plugin that handles the garbage loot from farming.
The plugin was created and is currently handled by GlitchToidZ a.k.a. GToidZ.

Commands

"/scrap" or "/syg"

Main command, used for showing help message.

/scrap money

Open GUI for selling via command.

/scrap exp

Open GUI for scraping via command.

/scrap reload

Admin Command
Reload all configuration files in plugin.

Permissions

syg.command.money

Allows player to open GUI for selling via command.

syg.command.exp

Allows player to open GUI for scraping via command.

syg.bypass

Allows player to use Scrap Your Gear in creative mode.

syg.admin

Allows player to access admin commands.

Configuration

This section is about the files in the data folder. All server owners and moderators should read this section.

config.yml

debug: Boolean

This option is to be enabled if you wanted to understand what's behind the process.

blocks.sell-block: String

Set the block for selling when player's interacted with.
**Use Minecraft namespaced key!

blocks.exp-block: String

Set the block for scraping when player's interacted with.
**Use Minecraft namespaced key!

gui.sell-gui.decorations: String
gui.sell-gui.info: String
gui.sell-gui.button: String

Set the items for selling GUI.
**Use Minecraft namespaced key!

gui.exp-gui.decorations: String
gui.exp-gui.info: String
gui.exp-gui.button: String

Set the items for scraping GUI.
**Use Minecraft namespaced key!

gui.exp-gui.bottle-enable: String
gui.exp-gui.bottle-disable: String

Set the items for toggling reward as EXP and XP Bottles.
**Use Minecraft namespaced key!

Formulas

Formulas in configuration are parsed with a library "exp4j," a Java library for evaluating mathematical expressions up to scientific functions. Please consult the variables by reference below. You might also want to reference the expression format from the official exp4j documentation.

material-formula: String

Will evaluate the value via material of the item.
<var1> : value of material from potentials.yml

durability-formula: String

Will evaluate the value via durability of the item.
<var1> : value of percentage durabilty from potentials.yml

enchantment-formula: String

Will evaluate the value via each enchantments of the item.
<var1> : value of a single enchantment from potentials.yml
<var2> : level of that enchantment.

total-formula: String

Will evaluate the final reward from using Scrap Your Gear.
<var1> : value from material evaluation.
<var2> : value from durability evaluation.
<var3> : value from all enchantment evaluations summed up.

bottle-divisor: Integer

When "Prefer Bottles" is enabled, this value will divide the Total Formula of EXP procedure.
**Only available in Scraping procedure.

potentials.yml

material.[smth]: Double

Determine each material with different values.

durabilities.[smth]: Double

Determine each durability ranges with different values.

enchantment.[smth].[money/exp]: Double

Determine each enchantment with different values.
Money and EXP are seperated as individual values.
(Optional) Adding "negative: true" option makes the enchantment lower the final reward.

messages.yml

chat.[message]: String

Self-explanatory, messages are formatted by "prefix" followed by the message queried.

gui.title: String

Title for Scrap Your Gear GUI.
